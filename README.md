# medical-insurance-eda-ml
# Medical Insurance Cost Analysis

## Problem Statement
Analyze factors affecting medical insurance charges using statistical
and statistical hypothesis testing methods.

## Dataset
Medical insurance dataset containing age, BMI, smoking status,
region, children, and insurance charges.

## Steps Performed
- Data cleaning and preprocessing
- Feature engineering (BMI categories, encoding)
- Pearson correlation analysis
- Chi-square test for categorical features
- Feature selection insights

## Tech Stack
- Python
- Pandas, NumPy
- SciPy
- Scikit-learn
- Matplotlib, Seaborn
- Jupyter Notebook

pip install -r requirements.txt
